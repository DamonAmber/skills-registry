# 耗材多端展示规则

本文件说明耗材在不同终端的展示差异，特别是 CarIoT 设备的特殊规则。

## 一、多端展示矩阵

| 终端 | 非 CarIoT 设备耗材 | CarIoT 设备耗材 |
|------|-------------------|----------------|
| 手机（米家 App） | ✅ | 不支持 |
| 插件（米家 App） | ✅ | ✅ |
| Pad（米家 App） | ✅ | 不支持 |
| 电视（米家 App） | ✅ | 不支持 |
| 带屏音箱 | ✅（部分型号） | 不支持 |
| 智能设备（车机） | 不支持 | ✅ |
| 米家 App（车机） | ✅ | 不支持 |

## 二、CarIoT 设备耗材特殊规则

### 2.1 展示范围

- **仅在车机和米家 App 车房间插件中展示**
- **不在米家 App 首页仪表盘和「我的-设备耗材」中展示**

### 2.2 设计原因

车载设备主场景在车内，不在通用管理页面展示。

### 2.3 示例

智能香氛机（车房间插件自行实现，非通用方案）

## 三、车房间说明

- 当有汽车接入米家时，会出现特殊的 `车房间`
- 车房间作为独立空间存在，但不归属于任何一个家庭
- 车房间内的设备耗材遵循 CarIoT 特殊规则
