# 耗材用户交互

本文件说明耗材支持的三种用户交互方式。

## 一、三种交互方式

| 交互 | 可选性 | 说明 |
|------|--------|------|
| 手动重置 | 可选支持 | 由设备厂商在小米开放平台配置。主要针对设备无法自动获取耗材状态的情况（如部分空调滤网）。用户点击【重置】后，重新开始计算该耗材的剩余使用量和周期 |
| 购买 | 可配置支持 | 跳转购买，当前支持：有品（已废弃）、小米商城（主要路径）、微信小程序 |
| 更换教学 | 可配置 | 视频教学或图文教学（最多 5 步） |

## 二、手动重置

### 2.1 适用场景

设备无法自动获取耗材状态，需要用户手动确认更换后重置计量。

### 2.2 典型示例

部分空调滤网、部分空气净化器滤芯。

### 2.3 配置

厂商在开发者平台配置"是否支持重置"（必填项）。

## 三、购买

### 3.1 购买链接配置

- 配置项：耗材链接（选填）
- 支持平台：有品（已废弃）、小米商城（主要路径）、微信小程序
- 配置后效果：耗材详情页显示「立即购买」按钮

### 3.2 组合购买（2026 新增）

- 来源：小米商城需求，推广耗材卡权益
- 入口位置：首页耗材弹窗、设备耗材页、耗材详情页
- 实现方式：米家侧提供入口，点击后调起小米商城 SDK 弹窗
- 设计原则：弱提示，不破坏耗材管理的核心体验

## 四、更换教学

### 4.1 配置

- 配置项：耗材更换教学（选填）
- 支持形式：视频教学 / 图文教学
- 图文教学限制：最多 5 步

### 4.2 展示

配置后，耗材详情页增加教学入口。
