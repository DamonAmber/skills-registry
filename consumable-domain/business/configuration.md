# 耗材配置逻辑

本文件说明耗材从配置到上线的完整链路，涉及开发者平台和 Admin 平台两个系统。

## 一、配置平台

### 1.1 开发者平台（厂商侧）

配置路径：创建产品 → 高阶配置 → 耗材配置 → 编辑耗材 → 多语言配置 → 消息推送配置

厂商在此完成耗材的基础定义和展示配置。

### 1.2 Admin 平台（小米侧）

| 场景 | 路径 |
|------|------|
| 非标准模板产品 | 平台管理 → 配置项标准化规则 → 耗材配置规则 |
| 品类模板产品 | 平台管理 → 标准化管理 → 品类标准模板 → 标准耗材 |
| 耗材审核 | 审核管理 → 上线米家 App → 耗材审核 / 耗材列表 |

## 二、耗材配置项

每个耗材对应以下配置：

### 2.1 属性上报配置

| 配置项 | 可编辑 | 说明 |
|--------|--------|------|
| 服务 | 不可改 | 关联的 Spec 服务 |
| 不足/耗尽状态属性 | 不可改 | 关联的 Spec Property |
| 属性值取值说明 | 不可改 | 属性值范围描述 |
| 不足状态定义 | 必填 | 厂商设置临界值 |
| 耗尽状态定义 | 选填 | 厂商设置临界值 |

### 2.2 App 展示配置

| 配置项 | 必填 | 约束 |
|--------|------|------|
| 耗材名称 | 是 | 15 字以内 |
| 耗材图片 | 是 | 默认/自定义，963×474px，PNG，≤500KB |
| 建议更换周期 | 否 | 配置后显示预估使用天数 |
| 耗材型号 | 是 | 20 字以内，帮助用户搜索 |
| 功能介绍 | 否 | 100 字以内 |
| 耗材链接 | 否 | 仅支持有品/小米商城链接，配置后显示「立即购买」 |
| 是否支持重置 | 是 | 布尔值 |
| 更换教学 | 否 | 视频/图文，最多 5 步 |

### 2.3 预估使用天数计算

当配置了"建议更换周期"时：

```
预估使用天数 = 剩余比例 × 建议更换周期
```
