# 耗材专业术语

本文件定义耗材领域的专有名词，避免沟通歧义。

| 术语 | 定义 | 备注 |
|------|------|------|
| 耗材 | 有余量上报、会耗尽、需更换/补充的设备结构或补充剂 | 米家 App 特定定义 |
| Spec-V2 | 小米 IoT 设备属性协议第二版 | 耗材仅支持 Spec-V2 |
| Spec 属性（Property） | 设备上报的属性值，耗材余量依赖于此 | 厂商在开发者平台关联 |
| 临界值 | 厂商设置的阈值，用于定义耗材"不足"和"耗尽"状态 | 不足必填，耗尽选填 |
| 设备插件 | 设备详情页，内部不称"单设备详情页" | 分标准插件和非标准插件 |
| 设备耗材页 | "我的"Tab 下的全量耗材列表页 | 路径：我的 → 设备耗材 |
| 耗材详情页 | 单个耗材的详细信息页，由 Native 通用页面渲染 | 插件通过传参调起 |
| 首页仪表盘 | 首页（米家）Tab 的仪表盘区域 | 仅展示不足/耗尽耗材 |
| 耗材卡片 | 设备插件中展示耗材状态的卡片组件 | 倾向标准化使用 Native 通用页 |
| 手动重置 | 用户手动重置耗材计量，重新开始计算余量 | 针对设备无法自动获取状态的情况 |
| 耗材卡 | 小米商城的耗材订阅权益卡，购买可享折扣 | 商城侧产品 |
| CarIoT | 车载 IoT 设备 | 耗材展示有特殊规则 |
| 品类标准模板 | Admin 平台中按品类预定义的标准耗材配置 | 模板产品使用 |
| 有品链接 | 小米有品商城的商品链接 | 已废弃，现主要用小米商城链接 |
